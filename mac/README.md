### Mac mic-mute-python
A simple utility to mute mic on MacOS

#### Pre-requisites
- python3.8 

#### Quick Run
- `cd mac`
- `pip install -r requirements.txt`
- `python app.py`

#### Build
- `cd mac`
- `pip install -r requirements.txt`
- `pyinstaller --onefile app.py`
- `python setup.py build`
- `cd dist`
- Here, you can use the `app` file as a regular desktop app
